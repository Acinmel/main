# 口播重制 · 产品说明

**一句话**：把对标爆款口播里的「说什么」抽出来、改成你的话，再配上 **你的专属数字人形象**，在一条流程里完成从参考视频到成片预览与任务跟进——面向短视频口播创作者与运营团队。

---

## 核心功能

- **专属数字人**：上传本人照片，选择风格，生成适合口播出镜的参考形象，后续口播制作与任务统一使用该形象。
- **口播文案获取与编辑**：粘贴视频链接（常见为短视频平台分享链接），拉取视频信息；将视频中的说话内容转为可编辑的口播文案，便于你改写、润色。
- **口播制作工作台**：在同一页面管理「参考视频 / 口播稿 / 口播形象照片」，支持生成演示成片预览（在已配置相关能力时），再进入正式任务流程。
- **任务与作品**：创建任务后，按步骤完成文案提取、改写、生成设置与进度查看，并在「我的作品」中管理产出。
- **账号体系**：注册与登录，数据按账号区分。

---

## 典型使用场景

| 场景 | 说明 |
|------|------|
| **跟拍爆款结构** | 找到同赛道爆款口播，快速拿到口播全文，改成自己的产品与话术，减少从零写稿的时间。 |
| **统一人设出镜** | 先做好数字人形象，多期口播沿用同一「脸」，风格一致，适合个人 IP 与品牌账号。 |
| **团队内容协作** | 运营负责找对标与改稿，制作按任务推进，结果在作品列表中沉淀，便于复盘与复用。 |
| **私有化交付** | 适合希望数据与业务留在自有环境的企业或机构（具体交付形态与范围以合同为准）。 |

## 价值与边界（面向采购与决策）

**价值**

- 把「找对标 → 拿文案 → 改稿 → 用我的人设出片」串成一条路径，降低口播类短视频的启动成本。
- 浏览器内完成主要操作，用户侧无需安装专业剪辑或开发工具。

**边界**

- 涉及第三方平台视频与账号能力时，须遵守平台规则与当地法律法规；具体能力是否可用以实际环境与平台政策为准。
- 数字人生成、成片预览、文案润色等智能化能力依赖所选服务与配置，效果与费用因方案而异。
- 部分流程在持续迭代中，售前建议以演示环境或试用为准。
